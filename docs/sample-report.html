Top 3 Risks

1. Security posture: SOC 2 non-compliant, 3 critical CVEs unpatched
No SOC 2 Type II. Penetration test (Nov 2025) identified 3 critical and 8 high-severity vulnerabilities. IAM relies on shared service accounts for production database access. Estimated remediation: £285K over 6 months.
2. Key-person dependency: 2 engineers hold 68% of core routing engine knowledge
The route optimisation algorithm (Meridian's core IP) was built by two senior engineers who have been with the company since founding. No documentation. No tests on critical paths. Bus factor = 2. Mitigation requires £120K investment in documentation, pair programming, and knowledge transfer over 4 months.
3. Architecture scalability ceiling at ~3.8x current load
The monolithic API handles 2.1M daily calls on vertically-scaled AWS instances. Horizontal scaling is not supported without architectural refactoring. At the value creation plan's projected growth rate, the ceiling is reached in Month 18. Refactoring estimate: £440K over 12 months.

Top 3 Value Creation Opportunities

1. Cloud cost optimisation: £210K annual savings (currently 2.3x industry benchmark)
AWS spend is £68K/month (£816K/year). Industry benchmark for this revenue band: £25K–30K/month. Key drivers: oversized RDS instances, no reserved capacity, 40% of S3 storage is orphaned data. 60-day implementation.
2. AI-powered route optimisation: £2.1M incremental ARR opportunity
Current routing algorithm is rule-based (2019 vintage). ML-based routing using existing data could improve accuracy by 12–18%, enabling premium tier pricing. Data quality is sufficient. Estimated build: £380K over 9 months. Payback: 7 months.
3. Data monetisation: benchmark dataset across 340+ logistics operators
Meridian processes routing and cost data from 340+ logistics companies. This dataset has intrinsic value as an industry benchmark product (fuel cost trends, route efficiency comparisons, seasonal patterns). Estimated development: £180K. Potential ARR: £600K–1.2M within 24 months.

3. Architecture & Infrastructure

Score: 4/5 (Good) — Modern stack with solid fundamentals. Key concern: monolithic API limits horizontal scaling.

Current Architecture

Meridian runs a hybrid architecture: a Django monolith handling the core routing engine and client API, plus four Node.js microservices (notifications, billing, data export, webhook relay). All services run on AWS EC2 in eu-west-2 (London). The frontend is a React/Next.js application served via CloudFront.

Scalability Assessment

Current architecture handles 2.1M daily API calls on two vertically-scaled EC2 instances (r5.4xlarge). Load testing indicates the ceiling at approximately 8M daily calls (~3.8x current). At the value creation plan's projected 40% annual growth, this ceiling is reached in approximately Month 18. Horizontal scaling requires extracting the routing engine from the Django monolith into a dedicated service with stateless request handling.

Appendix: Methodology

Assessment Framework

This report follows the Intregis 7-Dimension Technology Due Diligence Framework. Each dimension is assessed against standardised criteria and scored on a 1–5 scale:

ScoreLabelDefinition
1CriticalImmediate risk to business operations. Emergency intervention required.
2SignificantMajor gaps impacting value creation plan. 6-month remediation.
3ModerateFunctional but suboptimal. 12-month improvement roadmap.
4GoodMinor gaps. Optimisation opportunities exist.
5ExcellentBest-in-class for company size and stage.

Data Sources

  • CI/CD pipeline review (GitHub Actions)

  • Architecture documentation (Confluence, partial)
  • Organisation chart and team data (HR system export)
  • Financial data: technology spend breakdown (CFO-provided)

  • Stakeholder interviews: CEO, VP Engineering, CPO, 2 team leads, DevOps lead (6 sessions, 8 hours total)

Limitations

This assessment was conducted over 10 working days based on information provided by Meridian's management team and direct observation of systems. Intregis did not conduct independent penetration testing (relying on the NCC Group report) or audit financial statements. Findings reflect conditions as of the assessment date.
